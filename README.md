# Letterboxd List Scraper

A tool for scraping Letterboxd lists from a URL into CSV or JSON.

## Usage

### Install

```bash
git clone <repo>
cd Letterboxd-list-scraper
pip install -r requirements.txt
```

### Run

```bash
python -m listscraper <list-url>
```

### Example

```bash
python -m listscraper https://letterboxd.com/username/list/my-list/
```
